## Site Tom Troc

## Pour utiliser ce projet :

- Commencer par cloner le projet.
- Installez le projet chez vous, dans un dossier exécuté par un serveur local (type MAMP, WAMP, LAMP, etc...)
- Une fois installé chez vous, créez un base de données vide appelée : "site_tomtroc".
- Importez le fichier _site_tomtroc.sql_ dans votre base de données.

## Pour lancer le projet :

- Pour la configuration du projet, copier le fichier Cres_Cyprien_Option_A_2_config_022026.php et renommer la copie en _config.php_ et placée la dans le dossier _config_.
- Si nécessaire changer le DB_USER et le DB_PASSWORD dans le fichier _config.php_.
- Lancez le projet en local, dans le terminal de commande de votre IDE tapez la commande suivante : `php -S localhost:8080` (attention à être dans le dossier du projet avant de taper cette commande).
- Utilisez l'URL suivante : http://localhost:8080/
- Si ce port est déjà utilisé, vous pouvez en choisir un autre (ex : 8000) et utiliser l'URL suivante : http://localhost:8000/
- Pour vous connecter en partie utilisateur, le login est "cyprien@gmail.com" et le mot de passe est "qwerty".
